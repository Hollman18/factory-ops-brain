# Report schedules by role

Periodic reports belong here, not in the heartbeat.

## Principles
- Reports are periodic and role-based.
- Use the user profile to decide which schedules are active.
- Keep content aligned to role needs.
- Avoid over-reporting.

## Gerente
**Recommended schedules**
- Weekly report
- Monthly report

**Weekly report should include**
- OEE summary and change vs previous week
- best/worst-performing area/line/reference
- main increase/decrease drivers
- risk to monthly target
- executive recommendation

**Monthly report should include**
- OEE and KPI comparison vs previous month
- trend direction
- main improvements and deteriorations
- likely causes
- target compliance or risk
- recommended management focus

## Directivo
**Recommended schedules**
- Weekly report
- Monthly report

**Focus**
- strategic view
- target compliance
- major operational and quality risks
- areas requiring intervention or investment attention

## Supervisor
**Recommended schedules**
- Shift-close report
- Daily report

**Shift-close report should include**
- line/shift performance summary
- biggest losses
- main deviation vs previous comparable shift
- probable cause
- immediate action for next shift

**Daily report should include**
- area/line summary
- best/worst shift or line
- top issues of the day
- recommendation for next day

## Operador
**Recommended schedules**
- None by default
- Optional targeted alerts only if explicitly enabled

**Reason**
Operators usually benefit more from on-demand guidance than recurring summaries.

## Mantenimiento
**Recommended schedules**
- Daily technical report
- Optional weekly criticality summary

**Daily report should include**
- machines with highest risk
- deterioration or anomaly signals
- downtime/failure indicators
- intervention priority

**Weekly summary should include**
- recurring equipment issues
- trend in risk/deterioration
- recommended preventive focus

## Calidad
**Recommended schedules**
- Daily quality report
- Optional weekly quality summary

**Daily report should include**
- rejection/scrap/merma/SPC summary
- worst references/processes
- likely cause
- containment recommendation

**Weekly summary should include**
- trend in quality stability
- recurring deviations
- references/processes needing intervention
