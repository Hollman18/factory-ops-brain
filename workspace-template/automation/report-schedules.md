# report-schedules.md

Use role-based report defaults:
- gerente/directivo -> weekly/monthly, including relevant deviations and target/business risk
- supervisor/jefe/líder -> shift-close/daily, including operational deviations
- mantenimiento/confiabilidad -> daily technical + alert-driven, including technical deviations
- calidad -> daily quality + alert-driven, including quality deviations
- operador -> on-demand unless explicitly configured otherwise

Allow user customization of cadence, focus, and detail.
