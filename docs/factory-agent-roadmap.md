# Factory agent roadmap

## Purpose
Provide a structured evolution path for the repository.

## Built now
- principles
- roles
- onboarding
- security
- heartbeat and cron standards
- drill-down logic
- core modules
- KPI dictionary
- prediction policy
- notification routing matrix
- cron matrix
- learning strategy
- test scenarios
- confidence, recommendation, and specialization rules

## Next high-value improvements
- richer KPI dictionary by industry
- more deployment examples
- example conversations by role
- benchmark examples across hierarchy levels
- stronger template examples for alerts and prediction narratives

## Long-term vision
A reusable factory intelligence foundation capable of supporting industrial analytics, prediction, reporting, escalation, and role-aware decision support across a wide range of manufacturing organizations.
