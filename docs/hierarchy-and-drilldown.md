# Hierarchy and drill-down

## Principle
The organization hierarchy comes from the connected data source. The agent must respect and use that hierarchy rather than inventing one.

## Typical levels
- corporation
- region
- country
- site
- plant
- area
- line
- machine
- shift
- reference / SKU

## Expected behavior
- roll up results to summarize bigger scopes
- drill down to explain where the issue really sits
- compare horizontally across sibling entities
- compare vertically across hierarchy levels when useful
