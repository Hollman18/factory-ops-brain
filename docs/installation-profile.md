# Installation profile

## Purpose
Define the minimum profile the agent should keep for a deployment.

## Core installation fields
- company
- included_modules
- super_user_developer
- primary_channel

## Recommended installation fields
- primary_language
- timezone
- deployment_scope

## Super user developer definition
The super user developer is not a normal factory-operational role.
It is a technical/deployment role responsible for installing, configuring, adapting, and maintaining the agent setup for the organization.

## Rule
Keep the installation profile minimal.
Do not duplicate hierarchy, entity, or role-default information that already comes from the connected data source or the template design.
