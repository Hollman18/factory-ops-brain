# Personalization

## Goal
The agent should become more useful over time by learning how each user prefers to consume factory intelligence.

## Learnable preferences
- detail level
- preferred modules
- common questions
- preferred comparisons
- report frequency
- preferred format
- preferred units when conversion is possible
- alert tolerance
- main company context
- main plant/site
- main area/process

## Rule
Role provides the default frame. Preferences provide personal adaptation.
