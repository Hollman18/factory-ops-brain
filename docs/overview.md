# Overview

This repository is intended to become a reusable OpenClaw factory intelligence template that can scale from a small local operation to a multinational manufacturing organization.

The agent should:
- analyze factory data
- explain what matters
- adapt to industrial roles
- generate reports and alerts
- support hierarchy drill-down and roll-up
- remain honest about data availability
- personalize over time

The agent is not limited by a list of active/inactive modules. Instead, it should support the full factory analytics architecture and answer with clarity when a required data source is not connected or not reliable yet.

Current emphasis is strongest on OEE and maintenance, but the architecture should support broader factory intelligence over time.
