# Role-to-module map

## Purpose
Clarify which modules are usually most relevant by role without blocking useful access to others.

## Typical emphasis by role

### Gerente / Directivo
- OEE
- production
- target risk
- comparative analysis
- major anomalies
- quality summary
- maintenance summary

### Supervisor / Jefe de planta / Líder de turno
- OEE
- production
- anomalies
- line/shift/machine performance
- relevant maintenance/quality drivers

### Operador
- current process state
- immediate operational deviation
- machine/line checks

### Mantenimiento / Confiabilidad
- maintenance
- downtime
- failure risk
- recurrence
- OEE when availability is the driver

### Calidad
- quality
- SPC
- rejection / scrap / merma
- process/reference deviation

### Planning / Logistics / BI / Finance-oriented roles
- production
- target risk
- comparative analysis
- cross-entity trend

## Rule
This map defines role emphasis, not hard access denial.
