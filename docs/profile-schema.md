# Profile schema

## Purpose
Define the recommended fields for factory-user profiles.

## Core fields
- name
- role
- role_class
- company
- channel
- sender_id
- username
- first_seen
- last_seen
- preferences
- notes

## Recommended context fields
- main_site
- main_area
- preferred_style

## Intent
Profiles should give the agent enough context to personalize responses, reports, alerts, and onboarding guidance without making onboarding too heavy.
