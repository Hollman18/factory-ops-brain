# Deployment maturity model

## Purpose
Provide a realistic path from first deployment to advanced factory intelligence maturity.

## Stage 1 - Foundational intelligence
- onboarding by role
- security boundaries
- OEE and maintenance interpretation
- basic heartbeat alerting
- cron by role
- honest missing-data behavior

## Stage 2 - Structured operational intelligence
- stronger comparative analysis
- target risk
- better alert routing
- prediction policy
- KPI dictionary
- learning strategy

## Stage 3 - Expanded industrial coverage
- quality
- SPC
- energy
- raw materials
- stronger hierarchy usage
- richer personalization

## Stage 4 - Advanced enterprise operation
- cross-site benchmarking
- industry specialization
- confidence-aware recommendation quality
- refined severity/routing behavior
- broader deployment standardization

## Rule
Not every deployment needs to start at the same maturity stage. The template should support growth without redesign.
